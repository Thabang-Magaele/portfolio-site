FuelTrack - Fuel Administration System

FuelTrack is a fully developed fuel administration
and “get now, pay later” system built using Java,
JavaFX, SceneBuilder, SQLite, XML, and the
BulkSMS API, designed to streamline fuel requests
and approvals between users and administrators.
The system features secure user authentication
with role-based access, allowing users to submit
fuel requests that are stored in the SQLite
database, while administrators manage, approve,
or reject these requests through a clean JavaFX
interface. Every new request automatically
triggers an SMS notification to administrators
via the BulkSMS API, ensuring real-time response
and oversight. XML files are used for configuration
and structured system data, and the overall solution
provides a fast, organized, and automated workflow
that replaces manual fuel management with a modern, efficient digital process.
