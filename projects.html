FuelTrack — Fuel Administration System

A prototype fuel administration system that streamlines fuel requests and approvals between users and administrators. It features secure, role-based authentication — users submit requests stored in an SQLite database, while administrators manage, approve or reject them through a clean JavaFX interface.

Every new request automatically triggers an SMS notification to administrators via the BulkSMS API, ensuring real-time oversight. The result is a fast, organized and automated workflow that replaces manual fuel management.
